Below is a **Getting Started Guide** for using PyChemicals. This guide walks you through cloning the repository, installing the package, setting up your chemical data, and loading an example acid to calculate its pH.

---

## 1. Cloning the Repository

Clone the PyChemicals GitHub repository:

```bash
git clone https://github.com/Shoaib-Programmer/PyChemicals.git
cd PyChemicals
```

## 2. Installation

Install the dependencies

```bash
pip install -r requirements.txt
```

<br></br>
Now, you have two options to install the package:

### Standard Installation

Install the package in your environment by running:

```bash
pip install .
```

### Editable Installation (For Development)

If you plan on modifying the code or contributing, install in editable mode:

```bash
pip install -e .
```

This installs the package and allows you to update the code without reinstalling.

## 3. Database Setup

PyChemicals decouples chemical data from its computational logic by loading data from an external SQLite database. You can use PyChemicals's database, but if you want your own database, follow this schema:

### Acids Table

```sql
CREATE TABLE acids (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT UNIQUE,
    proticity INTEGER,
    Ka REAL,
    molar_mass REAL,
    ka1 REAL,
    ka2 REAL
);
```

### Bases Table

```sql
CREATE TABLE bases (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT UNIQUE,
    proticity INTEGER,
    Kb REAL,
    molar_mass REAL,
    kb1 REAL,
    kb2 REAL
);
```

### Gases Table

```sql
CREATE TABLE gases (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT UNIQUE,
    molar_mass REAL,
    density REAL
);
```

You can use the following types of files for the data:
`.db`, `.sqlite3`, `.sql`, and `.csv`.
Populate them with your chemical data. Then, set the data source using the `use_source` option:

```python
import PyChemicals as pc

pc.use_source('data.csv') # Path to your data source
```

## 4. Example usage

The chemicals library automatically loads the chemicals data from the set source.
Here's an example of how to load data for an acid and calculate its pH:

```python
from PyChemicals.chemicals import Acid

# Create an instance of Acid with concentration (in M) and volume (in L).
hcl = Acid("Hydrogen chloride", concentration=0.1, volume=0.025)

# Calculate and print the pH of the acid solution.
print("pH of Hydrogen chloride:", hcl.ph())
```
However, you can override the default data for a chemical by using the properties argument. For example:

```python
hcl = Acid("Hydrogen chloride", concentration=0.1, volume=0.025, properties={
    ... # Properties you want to override: refer to the schema on information about how this should be structured.
})
```

This snippet demonstrates:

- Loading the chemical properties from the PyChemicals database (use the `use_source` option to override this).
- Creating an `Acid` instance with a given concentration and volume.
- Computing and printing the pH.

By following these steps, you'll have PyChemicals installed and ready to perform various chemical calculations. As you progress, you can explore additional modules (like titration calculations and titration curves) that build on this foundation.
